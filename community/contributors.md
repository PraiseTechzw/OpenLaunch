## How This Page is Maintained

This contributors page is maintained through a combination of:

1. **Automated Updates**: GitHub Actions automatically update contributor lists based on repository activity
2. **Manual Recognition**: Community maintainers add special recognitions, awards, and spotlights
3. **Self-Nomination**: Contributors can nominate themselves or others for special recognition
4. **Community Input**: The community helps identify outstanding contributions and contributors

### Updating Your Information

If you'd like to update your information on this page:

1. **Profile Information**: Update your GitHub profile - it's automatically reflected here
2. **Recognition**: Nominate yourself or others through our recognition process
3. **Corrections**: Create an issue if you notice any errors or omissions
4. **Privacy**: Contact maintainers if you'd like to opt out of public recognition

### Recognition Process

To nominate someone (including yourself) for special recognition:

1. Create an issue using the "Contributor Recognition" template
2. Provide details about the contribution or achievement
3. Community maintainers will review and add appropriate recognition
4. Recognition is added during monthly contributor updates
